# SlopCode v0.1.39

This release makes starting a new session feel smoother and more predictable.

- `/new` now opens a single reusable draft tab in the top session strip.
- Your empty-state composer and any unsent prompt stay in place while you switch between tabs.
- Sending the first prompt turns that draft into a real session tab instead of resetting the strip.
- The tab keeps the `New Session` label until the generated session title is ready.
- Workspace and package versions are now updated to `0.1.39`.
