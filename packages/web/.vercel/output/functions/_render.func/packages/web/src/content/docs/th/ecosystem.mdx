ชุดโครงการชุมชนที่สร้างขึ้นบน SlopCode

:::note
ต้องการเพิ่มโครงการที่เกี่ยวข้องกับ SlopCode ของคุณลงในรายการนี้หรือไม่ ส่งพีอาร์.
:::

คุณยังสามารถลองดู [awesome-slopcode](https://github.com/awesome-slopcode/awesome-slopcode) และ [slopcode.cafe](https://slopcode.cafe) ซึ่งเป็นชุมชนที่รวบรวมระบบนิเวศและชุมชนไว้ด้วยกัน
