Una colección de proyectos comunitarios construidos en SlopCode.

:::note
¿Quieres agregar tu proyecto relacionado con SlopCode a esta lista? Envíe un PR.

También puedes consultar [awesome-slopcode](https://github.com/awesome-slopcode/awesome-slopcode) y [slopcode.cafe](https://slopcode.cafe), una comunidad que agrega el ecosistema y la comunidad.
