–Ъ–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—П

–Т—Л –Љ–Њ–ґ–µ—В–µ –љ–∞—Б—В—А–Њ–Є—В—М slopcode, –Є—Б–њ–Њ–ї—М–Ј—Г—П —Д–∞–є–ї –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є JSON.

–§–Њ—А–Љ–∞—В

slopcode –њ–Њ–і–і–µ—А–ґ–Є–≤–∞–µ—В —Д–Њ—А–Љ–∞—В—Л JSON –Є JSONC (JSON —Б –Ї–Њ–Љ–Љ–µ–љ—В–∞—А–Є—П–Љ–Є).

–†–∞—Б–њ–Њ–ї–Њ–ґ–µ–љ–Є–µ

–Т—Л –Љ–Њ–ґ–µ—В–µ —А–∞–Ј–Љ–µ—Б—В–Є—В—М —Б–≤–Њ—О –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—О –≤ –љ–µ—Б–Ї–Њ–ї—М–Ї–Є—Е —А–∞–Ј–љ—Л—Е –Љ–µ—Б—В–∞—Е, –Є —Г –љ–Є—Е –µ—Б—В—М —А–∞–Ј–љ—Л–є –њ–Њ—А—П–і–Њ–Ї —Б—В–∞—А—И–Є–љ—Б—В–≤–∞.

Note

–§–∞–є–ї—Л –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є –Њ–±—К–µ–і–Є–љ—П—О—В—Б—П, –∞ –љ–µ –Ј–∞–Љ–µ–љ—П—О—В—Б—П.

–§–∞–є–ї—Л –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є –Њ–±—К–µ–і–Є–љ—П—О—В—Б—П, –∞ –љ–µ –Ј–∞–Љ–µ–љ—П—О—В—Б—П. –Э–∞—Б—В—А–Њ–є–Ї–Є –Є–Ј —Б–ї–µ–і—Г—О—Й–Є—Е –Љ–µ—Б—В –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є –Њ–±—К–µ–і–Є–љ—П—О—В—Б—П. –С–Њ–ї–µ–µ –њ–Њ–Ј–і–љ–Є–µ –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є –њ–µ—А–µ–Њ–њ—А–µ–і–µ–ї—П—О—В –њ—А–µ–і—Л–і—Г—Й–Є–µ —В–Њ–ї—М–Ї–Њ –≤ —Б–ї—Г—З–∞–µ –Ї–Њ–љ—Д–ї–Є–Ї—В–∞ –Ї–ї—О—З–µ–є. –Э–µ–Ї–Њ–љ—Д–ї–Є–Ї—В–љ—Л–µ –љ–∞—Б—В—А–Њ–є–Ї–Є –Є–Ј –≤—Б–µ—Е –Ї–Њ–љ—Д–Є–≥–Њ–≤ —Б–Њ—Е—А–∞–љ—П—О—В—Б—П.

–Э–∞–њ—А–Є–Љ–µ—А, –µ—Б–ї–Є –≤–∞—И–∞ –≥–ї–Њ–±–∞–ї—М–љ–∞—П –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—П —Г—Б—В–∞–љ–∞–≤–ї–Є–≤–∞–µ—В theme: "slopcode" –Є autoupdate: true, –∞ –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—П –≤–∞—И–µ–≥–Њ –њ—А–Њ–µ–Ї—В–∞ —Г—Б—В–∞–љ–∞–≤–ї–Є–≤–∞–µ—В model: "anthropic/claude-sonnet-4-5", –Њ–Ї–Њ–љ—З–∞—В–µ–ї—М–љ–∞—П –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—П –±—Г–і–µ—В –≤–Ї–ї—О—З–∞—В—М –≤—Б–µ —В—А–Є –њ–∞—А–∞–Љ–µ—В—А–∞.

–Я–Њ—А—П–і–Њ–Ї –њ—А–Є–Њ—А–Є—В–µ—В–∞

–Ш—Б—В–Њ—З–љ–Є–Ї–Є –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є –Ј–∞–≥—А—Г–ґ–∞—О—В—Б—П –≤ —Б–ї–µ–і—Г—О—Й–µ–Љ –њ–Њ—А—П–і–Ї–µ (–±–Њ–ї–µ–µ –њ–Њ–Ј–і–љ–Є–µ –Є—Б—В–Њ—З–љ–Є–Ї–Є –њ–µ—А–µ–Њ–њ—А–µ–і–µ–ї—П—О—В –±–Њ–ї–µ–µ —А–∞–љ–љ–Є–µ):

  1. –£–і–∞–ї–µ–љ–љ–∞—П –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—П (–Њ—В .well-known/slopcode) вАУ –Њ—А–≥–∞–љ–Є–Ј–∞—Ж–Є–Њ–љ–љ—Л–µ –љ–∞—Б—В—А–Њ–є–Ї–Є –њ–Њ —Г–Љ–Њ–ї—З–∞–љ–Є—О.
  2. –У–ї–Њ–±–∞–ї—М–љ–∞—П –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—П (~/.config/slopcode/slopcode.json) вАФ –љ–∞—Б—В—А–Њ–є–Ї–Є –њ–Њ–ї—М–Ј–Њ–≤–∞—В–µ–ї—П.
  3. –Я–Њ–ї—М–Ј–Њ–≤–∞—В–µ–ї—М—Б–Ї–∞—П –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—П (SLOPCODE_CONFIG env var) вАУ –њ–Њ–ї—М–Ј–Њ–≤–∞—В–µ–ї—М—Б–Ї–Є–µ –њ–µ—А–µ–Њ–њ—А–µ–і–µ–ї–µ–љ–Є—П
  4. –Ъ–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—П –њ—А–Њ–µ–Ї—В–∞ (slopcode.json –≤ –њ—А–Њ–µ–Ї—В–µ) вАФ –љ–∞—Б—В—А–Њ–є–Ї–Є, —Б–њ–µ—Ж–Є—Д–Є—З–љ—Л–µ –і–ї—П –њ—А–Њ–µ–Ї—В–∞.
  5. –Ъ–∞—В–∞–ї–Њ–≥–Є .slopcode вАФ –∞–≥–µ–љ—В—Л, –Ї–Њ–Љ–∞–љ–і—Л, –њ–ї–∞–≥–Є–љ—Л
  6. –Т—Б—В—А–Њ–µ–љ–љ–∞—П –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—П (SLOPCODE_CONFIG_CONTENT env var) вАУ –њ–µ—А–µ–Њ–њ—А–µ–і–µ–ї—П–µ—В –≤—А–µ–Љ—П –≤—Л–њ–Њ–ї–љ–µ–љ–Є—П

–≠—В–Њ –Њ–Ј–љ–∞—З–∞–µ—В, —З—В–Њ –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є –њ—А–Њ–µ–Ї—В–∞ –Љ–Њ–≥—Г—В –њ–µ—А–µ–Њ–њ—А–µ–і–µ–ї—П—В—М –≥–ї–Њ–±–∞–ї—М–љ—Л–µ –љ–∞—Б—В—А–Њ–є–Ї–Є –њ–Њ —Г–Љ–Њ–ї—З–∞–љ–Є—О, –∞ –≥–ї–Њ–±–∞–ї—М–љ—Л–µ –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є –Љ–Њ–≥—Г—В –њ–µ—А–µ–Њ–њ—А–µ–і–µ–ї—П—В—М –љ–∞—Б—В—А–Њ–є–Ї–Є –њ–Њ —Г–Љ–Њ–ї—З–∞–љ–Є—О –і–ї—П —Г–і–∞–ї–µ–љ–љ–Њ–є –Њ—А–≥–∞–љ–Є–Ј–∞—Ж–Є–Є.

Note

–Т –Ї–∞—В–∞–ї–Њ–≥–∞—Е .slopcode –Є ~/.config/slopcode –і–ї—П –њ–Њ–і–Ї–∞—В–∞–ї–Њ–≥–Њ–≤ –Є—Б–њ–Њ–ї—М–Ј—Г—О—В—Б—П –Љ–љ–Њ–ґ–µ—Б—В–≤–µ–љ–љ—Л–µ –Є–Љ–µ–љ–∞: agents/, commands/, modes/, plugins/, skills/, tools/ –Є themes/. –Х–і–Є–љ—Б—В–≤–µ–љ–љ—Л–µ –Є–Љ–µ–љ–∞ (–љ–∞–њ—А–Є–Љ–µ—А, agent/) —В–∞–Ї–ґ–µ –њ–Њ–і–і–µ—А–ґ–Є–≤–∞—О—В—Б—П –і–ї—П –Њ–±—А–∞—В–љ–Њ–є —Б–Њ–≤–Љ–µ—Б—В–Є–Љ–Њ—Б—В–Є.

–£–і–∞–ї–µ–љ–љ–∞—П

–Ю—А–≥–∞–љ–Є–Ј–∞—Ж–Є–Є –Љ–Њ–≥—Г—В –њ—А–µ–і–Њ—Б—В–∞–≤–Є—В—М –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—О –њ–Њ —Г–Љ–Њ–ї—З–∞–љ–Є—О —З–µ—А–µ–Ј –Ї–Њ–љ–µ—З–љ—Г—О —В–Њ—З–Ї—Г .well-known/slopcode. –Ю–љ –Є–Ј–≤–ї–µ–Ї–∞–µ—В—Б—П –∞–≤—В–Њ–Љ–∞—В–Є—З–µ—Б–Ї–Є –њ—А–Є –∞—Г—В–µ–љ—В–Є—Д–Є–Ї–∞—Ж–Є–Є —Г –њ—А–Њ–≤–∞–є–і–µ—А–∞, –Ї–Њ—В–Њ—А—Л–є –µ–≥–Њ –њ–Њ–і–і–µ—А–ґ–Є–≤–∞–µ—В.

–£–і–∞–ї–µ–љ–љ–∞—П –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—П –Ј–∞–≥—А—Г–ґ–∞–µ—В—Б—П –њ–µ—А–≤–Њ–є –Є —Б–ї—Г–ґ–Є—В –±–∞–Ј–Њ–≤—Л–Љ —Б–ї–Њ–µ–Љ. –Т—Б–µ –Њ—Б—В–∞–ї—М–љ—Л–µ –Є—Б—В–Њ—З–љ–Є–Ї–Є –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є (–≥–ї–Њ–±–∞–ї—М–љ—Л–µ, –њ—А–Њ–µ–Ї—В–љ—Л–µ) –Љ–Њ–≥—Г—В –њ–µ—А–µ–Њ–њ—А–µ–і–µ–ї–Є—В—М —Н—В–Є –Ј–љ–∞—З–µ–љ–Є—П –њ–Њ —Г–Љ–Њ–ї—З–∞–љ–Є—О.

–Э–∞–њ—А–Є–Љ–µ—А, –µ—Б–ї–Є –≤–∞—И–∞ –Њ—А–≥–∞–љ–Є–Ј–∞—Ж–Є—П –њ—А–µ–і–Њ—Б—В–∞–≤–ї—П–µ—В —Б–µ—А–≤–µ—А—Л MCP, –Ї–Њ—В–Њ—А—Л–µ –њ–Њ —Г–Љ–Њ–ї—З–∞–љ–Є—О –Њ—В–Ї–ї—О—З–µ–љ—Л:

–Т—Л –Љ–Њ–ґ–µ—В–µ –≤–Ї–ї—О—З–Є—В—М –Њ–њ—А–µ–і–µ–ї–µ–љ–љ—Л–µ —Б–µ—А–≤–µ—А—Л –≤ –ї–Њ–Ї–∞–ї—М–љ–Њ–є –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є:

–У–ї–Њ–±–∞–ї—М–љ–∞—П

–Я–Њ–Љ–µ—Б—В–Є—В–µ –≥–ї–Њ–±–∞–ї—М–љ—Г—О –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—О slopcode –≤ ~/.config/slopcode/slopcode.json. –Ш—Б–њ–Њ–ї—М–Ј—Г–є—В–µ –≥–ї–Њ–±–∞–ї—М–љ—Г—О –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—О –і–ї—П –Њ–±—Й–µ–њ–Њ–ї—М–Ј–Њ–≤–∞—В–µ–ї—М—Б–Ї–Є—Е –љ–∞—Б—В—А–Њ–µ–Ї, —В–∞–Ї–Є—Е –Ї–∞–Ї —В–µ–Љ—Л, –њ–Њ—Б—В–∞–≤—Й–Є–Ї–Є –Є–ї–Є –њ—А–Є–≤—П–Ј–Ї–Є –Ї–ї–∞–≤–Є—И.

–У–ї–Њ–±–∞–ї—М–љ–∞—П –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—П –њ–µ—А–µ–Њ–њ—А–µ–і–µ–ї—П–µ—В –љ–∞—Б—В—А–Њ–є–Ї–Є –њ–Њ —Г–Љ–Њ–ї—З–∞–љ–Є—О –і–ї—П —Г–і–∞–ї–µ–љ–љ–Њ–є –Њ—А–≥–∞–љ–Є–Ј–∞—Ж–Є–Є.

–Я—А–Њ–µ–Ї—В–љ–∞—П

–Ф–Њ–±–∞–≤—М—В–µ slopcode.json –≤ –Ї–Њ—А–µ–љ—М –≤–∞—И–µ–≥–Њ –њ—А–Њ–µ–Ї—В–∞. –Ъ–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є—П –њ—А–Њ–µ–Ї—В–∞ –Є–Љ–µ–µ—В –љ–∞–Є–≤—Л—Б—И–Є–є –њ—А–Є–Њ—А–Є—В–µ—В —Б—А–µ–і–Є —Б—В–∞–љ–і–∞—А—В–љ—Л—Е —Д–∞–є–ї–Њ–≤ –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є вАФ –Њ–љ–∞ –њ–µ—А–µ–Њ–њ—А–µ–і–µ–ї—П–µ—В –Ї–∞–Ї –≥–ї–Њ–±–∞–ї—М–љ—Л–µ, —В–∞–Ї –Є —Г–і–∞–ї–µ–љ–љ—Л–µ –Ї–Њ–љ—Д–Є–≥—Г—А–∞—Ж–Є–Є.
